# Glassmorphism

## Best For
- Modern SaaS
- Fintech dashboards
- Creative agencies
- Premium apps
- Portfolio sites

## Vibe
Translucent, frosted-glass effect with background blur. Premium, modern, depth-filled.

## Color Palette
- Backgrounds: Dark gradients or vibrant images
- Surface: `rgba(255,255,255,0.1)` with `backdrop-filter: blur(10px)`
- Borders: `rgba(255,255,255,0.2)`
- Text: White or light gray
- Accents: Vibrant blues, purples, pinks

## Typography
- Font: `Satoshi`, `Manrope`, `Plus Jakarta Sans`
- Medium weights
- Clear contrast against blurred backgrounds

## Components
- Blurred cards
- Subtle white borders
- Floating elements
- Gradient backgrounds

## When to Use
- Premium SaaS
- Dashboards
- Creative portfolios
- Modern apps

## When NOT to Use
- Trades/emergency services
- Budget brands
- High-accessibility requirements
- Elderly audiences
