# Soft UI Evolution

## Best For
- Wellness brands
- Beauty/spa
- Lifestyle services
- Premium personal services
- Yoga/meditation

## Vibe
Soft shadows, subtle depth, calming, premium feel. Organic shapes and gentle transitions.

## Color Palette
- Background: `#FFF5F5` or warm white
- Surface: `#FFFFFF`
- Primary: `#E8B4B8` (soft pink)
- Secondary: `#A8D5BA` (sage)
- Accent: `#D4AF37` (gold)
- Text: `#2D3436`

## Typography
- Font: `Montserrat`, `Cormorant Garamond`, `Outfit`
- Elegant serif headlines + clean sans body

## Components
- Soft rounded corners: `16px`
- Gentle shadows
- Organic shapes
- Smooth transitions

## When to Use
- Beauty/wellness
- Spa/salon
- Lifestyle brands
- Premium services

## When NOT to Use
- Industrial/trades
- Emergency services
- Budget brands
- Tech/developer tools
