# CTA Pattern: Final Call-to-Action Banner

**Source:** General pattern — section before footer
**Best for:** All conversion-focused sites
**Vibe:** Urgency, clear next step

## Description
Full-width section with strong headline, supporting text, and prominent CTA button. Last chance to convert before footer.

## Layout Structure
```
┌─────────────────────────────────────────────────────────────┐
│                                                             │
│              Ready to {ACTION}?                             │
│                                                             │
│         Brief supporting text that reinforces               │
│         the value proposition one more time.                │
│                                                             │
│              [PRIMARY CTA BUTTON]                           │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

## Visual Specs

**Background:**
- Brand color or dark color (contrasting)
- Or subtle gradient
- Full-width, edge to edge

**Headline:**
- Large, bold
- Action-oriented: "Ready to Get Started?"
- White or contrasting color

**Subheadline:**
- 1-2 lines
- Supportive, not repetitive
- Smaller text

**CTA Button:**
- Large, prominent
- Contrasting color to background
- Clear action text

## When to Use
- Before footer on every page
- After all content has made the case
- Final conversion opportunity

## Variations
- **With form:** Email input + button inline
- **With phone:** "Call Now: (555) 123-4567"
- **Split:** Content left, CTA right
- **Minimal:** Just headline + button

## Prompt Snippet
```
Final CTA Section: Conversion banner
- Background: {BG_COLOR} (brand color or dark)
- Layout: Centered content
- Headline: "{HEADLINE_TEXT}" — large (36-48px), bold, white
- Subheadline: "{SUBHEADLINE_TEXT}" — regular, white/light
- CTA Button: 
  * Text: "{CTA_TEXT}"
  * Style: {BUTTON_STYLE} (solid, contrasting color)
  * Size: Large, prominent
- Padding: 80-100px top/bottom
- Optional: Decorative elements (subtle shapes, patterns)
```

## Notes
- This is the final conversion point — make it count
- Don't be shy about the ask here
- Button should be impossible to miss
- Keep text concise — they've already read the whole page
