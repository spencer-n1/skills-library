# Navigation Pattern: Blurred Background with Gradient Text

**Source:** XPO Rehab (xpo-lead-pulse.base44.app)
**Best for:** Healthcare, wellness, modern tech, creative agencies
**Vibe:** Modern, sleek, contemporary

## Description
Navigation with frosted glass / blurred background effect. Gradient text treatment for the business name/logo. Clean, modern aesthetic.

## Layout Structure
```
[LOGO] ──────────────────────── [Menu Items] [CTA]
   ↓                                  ↓
gradient text                    blurred bg
```

## Visual Specs

**Background:**
- Transparent/frosted glass effect
- backdrop-filter: blur() effect
- Semi-transparent background color (white at 80% opacity or similar)
- Subtle shadow on scroll

**Logo:**
- Gradient text effect (brand colors)
- Example: Blue to purple gradient, or brand color fade
- Clean, modern sans-serif font

**Menu Items:**
- Right-aligned or evenly spaced
- Simple text links, no underline
- Hover: color change or subtle underline

**CTA:**
- Contrasting button (solid color on blurred bg)
- Or text link with arrow

**Behavior:**
- Sticky on scroll
- Background becomes more opaque on scroll (optional)
- Smooth blur transition

## When to Use
- Rehab/healthcare facilities
- Med spas, wellness centers
- Modern tech companies
- Creative agencies
- Any brand wanting contemporary feel

## Variations
- **Gradient logo:** Multi-color gradient text
- **Solid logo:** Single color with blur bg
- **Full blur:** Entire nav area frosted
- **Bottom blur:** Only bottom edge blurred (subtle)

## Prompt Snippet
```
Navigation: Blurred glass effect
- Position: Fixed/sticky at top
- Background: Semi-transparent with backdrop blur effect
- Logo: Gradient text using {PRIMARY_COLOR} to {SECONDARY_COLOR}
- Font: Modern sans-serif, medium weight
- Menu items: Right-aligned, clean text links
- CTA: Solid {CTA_COLOR} button for contrast
- On scroll: Background gains slight opacity
- Mobile: Hamburger menu with smooth animation
```

## Notes
- The blur effect creates depth and modern feel
- Gradient text must be subtle — not rainbow, brand colors only
- Contrast is important — ensure text remains readable over content
- Works best over images/video in hero section
