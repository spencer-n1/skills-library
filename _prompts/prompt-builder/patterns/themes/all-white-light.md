# Theme Pattern: All-White / Light Clean Aesthetic

**Source:** PureGlow Med Spa (curvy-pure-glow-path.base44.app)
**Best for:** Med spas, luxury wellness, modern healthcare, premium services
**Vibe:** Clean, airy, elegant, premium

## Description
White hero flowing into white/light sections throughout. No dark sections breaking the flow. Creates seamless, gallery-like, premium experience.

## Layout Structure
```
┌─────────────────────────────────────────────┐
│  [White Hero Section]                       │
│  White background, elegant imagery          │
├─────────────────────────────────────────────┤
│  [White Services Section]                   │
│  Subtle differentiation via cards/shadows   │
├─────────────────────────────────────────────┤
│  [Off-White/Light Section]                  │
│  Slight variation for visual rhythm         │
├─────────────────────────────────────────────┤
│  [White Section]                            │
│  Continues clean aesthetic                  │
├─────────────────────────────────────────────┤
│  [Footer - may be dark for contrast]        │
└─────────────────────────────────────────────┘
```

## Visual Specs

**Color Palette:**
- **Primary background:** Pure white (#ffffff) or warm white (#fafafa)
- **Secondary background:** Very light gray (#f8f9fa) or cream (#fefcf8)
- **Accent:** Soft brand color (blush, sage, soft blue) used sparingly
- **Text:** Dark gray or soft black (not pure #000), or charcoal

**Differentiation Between Sections:**
- Card shadows (subtle)
- Border dividers (hairline)
- Slight background tint shifts
- Spacing and padding
- NOT: Dramatic color changes

**Typography:**
- Often mixed (serif + sans) for elegance
- Generous whitespace
- Light, airy feel

**Imagery:**
- High-quality, bright, well-lit
- Consistent color grading (light, airy)
- Professional photography essential

## When to Use
- Med spas
- Luxury wellness
- Beauty/aesthetics
- High-end dentistry
- Wedding services
- Premium hospitality

## When NOT to Use
- Industrial/construction (too soft)
- Emergency services (needs urgency/dark contrast)
- Budget services (feels too expensive)
- Male-focused trades (may feel too feminine)

## Variations
- **Pure white:** #ffffff throughout, minimal variation
- **Warm white:** Cream, off-white tones for warmth
- **White with soft accent:** One soft color for CTAs/highlights
- **White to light gray:** Subtle progression for depth

## Prompt Snippet
```
Theme: All-white / light clean aesthetic
- Primary background: #ffffff (pure white) or #fafafa (warm white)
- Secondary sections: #f8f9fa (very light gray) or #fefcf8 (cream)
- Section differentiation:
  * Use subtle shadows on cards, not background color changes
  * Hairline borders between major sections if needed
  * Generous whitespace instead of color blocks
- Typography: Mixed serif/sans for elegance OR clean sans throughout
- Imagery: Bright, airy, professional photography with consistent grading
- Accent color: Soft {BRAND_COLOR} for CTAs only (blush, sage, soft blue)
- Text color: #1a1a1a (soft black) or #374151 (dark gray) — not pure #000
- Footer: Can introduce dark color for contrast and closure
- Overall feel: Gallery-like, seamless, premium, breathable
```

## Notes
- Requires EXCELLENT imagery — no room for error with white backgrounds
- Every element must be refined — white shows imperfections
- Contrast is still important — don't go too light on text
- This is Spencer's "white hero with rest of white website" pattern
- Feels expensive and curated when done right
