# AI SEO

Leverage AI tools to accelerate SEO content creation, optimization, and research while maintaining quality and originality.

## When to Use
- Scaling content production
- Keyword research and clustering
- Content brief creation
- First drafts and outlines
- Content optimization
- Meta tag generation

## Don't Use When
- YMYL (Your Money Your Life) topics requiring expertise
- Content needing original research/data
- Highly regulated industries without review
- When E-E-A-T signals are critical

## AI SEO Workflow

### Phase 1: Research & Strategy

**Keyword Clustering:**
- Input: Seed keywords
- AI groups related terms by intent
- Output: Content clusters with pillar + supporting pages

**Content Gap Analysis:**
- Analyze top-ranking content
- Identify subtopics they cover
- Find gaps your content can fill

**Search Intent Analysis:**
- Classify keywords (Informational, Navigational, Transactional, Commercial)
- Match content type to intent
- Identify SERP features to target

### Phase 2: Content Brief Creation

**Brief Elements:**
- Target keyword and secondary keywords
- Search intent classification
- Content type (guide, listicle, comparison, etc.)
- Target word count (based on top-ranking pages)
- Required sections/subtopics
- Questions to answer
- Internal linking opportunities
- Competitor URLs to reference

### Phase 3: Content Generation

**AI-Assisted Writing:**
1. Generate outline based on brief
2. Expand each section with AI
3. Add examples, case studies, data
4. Edit for voice and accuracy
5. Fact-check all claims

**Best practices:**
- Use AI for structure, add human expertise
- Never publish raw AI output
- Add original insights and examples
- Include personal experience (E-E-A-T)

### Phase 4: Optimization

**AI-Powered Improvements:**
- Title tag variations
- Meta description options
- Internal linking suggestions
- Readability improvements
- FAQ generation

## Prompts for SEO Content

### Content Outline
```
Create a detailed outline for an article targeting "[keyword]".
Search intent: [informational/commercial/etc.]
Include: H2s, H3s, key points to cover, estimated word count per section.
Analyze top 3 ranking pages and include topics they cover.
```

### Title Tags
```
Generate 10 title tag options for a page about "[topic]".
Target keyword: [keyword]
Max 60 characters each.
Include emotional triggers and power words.
```

### Meta Descriptions
```
Write 5 meta descriptions for "[page title]".
Include keyword: [keyword]
Max 160 characters.
Include clear value prop and CTA.
```

### Content Expansion
```
Expand this section to 300 words:
[section content]
Add examples, actionable tips, and maintain [tone].
```

## Quality Control Checklist

- [ ] Fact-checked all statistics and claims
- [ ] Added original insights or examples
- [ ] Included personal experience where relevant
- [ ] Reviewed for brand voice consistency
- [ ] Checked for AI hallucinations
- [ ] Optimized for target keywords naturally
- [ ] Added internal links
- [ ] Included compelling CTAs

- **Research:** Surfer SEO, Clearscope, MarketMuse
- **Optimization:** Frase, Dashword, Outranking
- **Technical:** Screaming Frog (AI features), Sitebulb
