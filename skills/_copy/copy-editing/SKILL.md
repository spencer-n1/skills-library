# Copy Editing

Transform rough drafts into polished, compelling copy through systematic editing.

## When to Use
- First drafts need refinement
- Copy isn't converting
- Messaging feels off
- Pre-publication review
- Refreshing existing content

## The Editing Process

### Round 1: Structural Edit
**Focus:** Big picture — does this accomplish the goal?

**Questions:**
- Is the message clear within 5 seconds?
- Does the structure flow logically?
- Is the CTA obvious and compelling?
- Is there enough proof/support?
- Does it match the reader's awareness level?

**Actions:**
- Reorder sections if needed
- Add missing proof points
- Cut tangents or weak sections
- Ensure one clear message

### Round 2: Line Edit
**Focus:** Clarity and flow — sentence by sentence

**Checklist:**
- [ ] Cut unnecessary words (aim for 10-20% reduction)
- [ ] Replace passive voice with active
- [ ] Shorten long sentences (break into 2)
- [ ] Replace jargon with plain language
- [ ] Add transitions between paragraphs
- [ ] Ensure "you" outnumbers "we" 2:1

**Common Word Replacements:**
| Instead of | Use |
|------------|-----|
| Utilize | Use |
| In order to | To |
| Due to the fact that | Because |
| At this point in time | Now |
| In the event that | If |
| A large number of | Many |

### Round 3: Copy Polish
**Focus:** Impact and persuasion

**Headlines:**
- Does it stop the scroll?
- Is the benefit clear?
- Would you click it?

**Openings:**
- First sentence must pull them in
- Start with the reader's problem or curiosity gap
- Cut throat-clearing ("In today's world...")

**CTAs:**
- Is the action clear?
- Does it reduce friction?
- Is there urgency or incentive?

**Specificity:**
- Replace vague claims with specifics
- "Grow faster" → "Increase conversions 34%"
- "Many customers" → "14,000+ customers"
- "Save time" → "Save 5 hours/week"

### Round 4: Proofreading
**Focus:** Error-free copy

**Checklist:**
- [ ] Spelling (don't trust spellcheck)
- [ ] Grammar (Grammarly, Hemingway)
- [ ] Punctuation consistency
- [ ] Number formatting (consistency)
- [ ] Brand name spellings
- [ ] Link functionality
- [ ] Mobile formatting

## Editing Techniques

### The So What Test
For every claim, ask "So what?" until you reach a real benefit.

Example:
- Feature: "We use 256-bit encryption"
- So what? "Your data is secure"
- So what? "Your customers trust you"
- **Benefit:** "Win customer trust with bank-level security"

### The 5-Second Test
Show someone the copy for 5 seconds, then ask:
- What do we do?
- Who is it for?
- What should you do next?

### Read Aloud
- Awkward phrasing becomes obvious
- Rhythm and flow issues surface
- You'll catch missing words

### Fresh Eyes
- Step away for at least an hour
- Edit in different format (print, different device)
- Read backwards (catches spelling errors)

## Common Issues & Fixes

| Issue | Fix |
|-------|-----|
| Too long | Cut 20%, focus on strongest points |
| Boring | Add specifics, stories, or surprising facts |
| Unclear | Simplify vocabulary, shorter sentences |
| Weak CTA | Add action verb, reduce friction words |
| No proof | Add testimonials, stats, or examples |
| Feature-heavy | Lead with benefits, mention features second |
