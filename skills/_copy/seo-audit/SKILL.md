# SEO Audit

Conduct comprehensive SEO audits to identify technical issues, on-page optimizations, and content opportunities.

## When to Use
- New site launch
- Traffic drop investigation
- Quarterly SEO health checks
- Before major site changes
- Competitive SEO analysis

## Audit Components

### 1. Technical SEO

**Crawlability & Indexing:**
- Robots.txt blocking important pages?
- XML sitemap present and submitted?
- Canonical tags correct?
- Noindex/nofollow issues?

**Site Speed:**
- Core Web Vitals (LCP, FID, CLS)
- Image optimization
- JavaScript/CSS minification
- Server response time

**Mobile-Friendliness:**
- Responsive design
- Touch targets
- Font sizes
- Mobile page speed

**URL Structure:**
- Clean, descriptive URLs
- Consistent structure
- No unnecessary parameters
- Proper redirects (301 vs 302)

**Security:**
- HTTPS implementation
- Mixed content issues
- Security headers

### 2. On-Page SEO

**Title Tags:**
- Unique per page
- 50-60 characters
- Primary keyword first
- Compelling click-through

**Meta Descriptions:**
- 150-160 characters
- Include primary keyword
- Clear value proposition
- Call-to-action

**Heading Structure:**
- One H1 per page
- Logical H2/H3 hierarchy
- Keywords in headings naturally

**Content Optimization:**
- Primary keyword in first 100 words
- Related keywords (LSI)
- Content depth (vs top-ranking pages)
- Internal linking

**Image Optimization:**
- Descriptive file names
- Alt text
- Compressed file sizes
- Responsive images

### 3. Off-Page SEO

**Backlink Profile:**
- Total referring domains
- Domain authority distribution
- Toxic/link spam check
- Anchor text distribution

**Brand Mentions:**
- Unlinked brand mentions (opportunities)
- Social signals

### 4. Content Audit

**Existing Content:**
- Thin content (under 300 words)
- Duplicate content
- Outdated content
- Content gaps vs competitors

**Keyword Cannibalization:**
- Multiple pages targeting same keyword
- Internal competition

- **Speed:** PageSpeed Insights, GTmetrix, WebPageTest
- **Rankings:** SEMrush, Ahrefs, Moz
