# Coastal Blue - Components

**Buttons:**
- Background: `#4A90B8`
- Text: `#FFFFFF`
- Border-radius: `8px`
- Hover: `#5BA3C8`

**Cards:**
- Background: `#FFFFFF`
- Border: `1px solid #D0E8F2`
- Border-radius: `12px`
- Shadow: none or very subtle

**Dividers:**
- Wave shapes at section boundaries
- Soft curves (not sharp angles)
