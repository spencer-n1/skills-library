# Linear Dark

**Source:** Linear.app

## Best For
- SaaS products
- Developer tools
- Project management apps
- Tech startups
- Dashboards

## Vibe
Ultra-minimal, precise, technical. Purple accent on dark. The gold standard for modern dark-mode SaaS interfaces.

## Color Palette
- Background: `#0E0E10`
- Surface: `#1C1C1F`
- Border: `#2A2A2D`
- Primary Text: `#FFFFFF`
- Secondary Text: `#8A8F98`
- Accent: `#5E6AD2`
- Accent Hover: `#7B82E0`

## Typography
- Font: `Inter` or geometric sans
- Tracking: tight
- Weight: medium for UI, semibold for headings

## Components
- Rounded corners: `6px`
- Subtle borders
- Glow accents on hover
- Clean icons

## When to Use
- SaaS/tech products
- Dashboards
- Developer-focused tools
- Modern B2B

## When NOT to Use
- Local service businesses
- Healthcare
- Elderly audiences
- Warm/personal brands
