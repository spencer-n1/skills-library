# Stripe Modern

**Source:** Stripe.com

## Best For
- Fintech
- SaaS
- Enterprise B2B
- Developer platforms
- High-growth startups

## Vibe
Clean, trustworthy, technically sophisticated. Gradient accents, precise illustrations, and impeccable spacing.

## Color Palette
- Background: `#FFFFFF`
- Surface: `#F6F9FC`
- Primary Text: `#0A2540`
- Secondary Text: `#425466`
- Accent: `#635BFF`
- Accent Hover: `#7A73FF`
- Success: `#00D4AA`

## Typography
- Font: `Inter` or `Söhne`
- Clean hierarchy
- Generous line-height

## Components
- Soft gradient buttons
- Card shadows
- Subtle grid patterns
- Rounded corners: `8px`

## When to Use
- Fintech
- Payments/ SaaS
- Enterprise
- Developer tools

## When NOT to Use
- Personal/local services
- Rustic/craft brands
- Budget/concrete trades
