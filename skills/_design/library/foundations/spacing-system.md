# Spacing System

## Rating
- **Status**: foundational
- **Preferred contexts**: all
- **Avoid contexts**: none

---

## Section Spacing

**Standard Section Padding:**
- Desktop: `py-20` (80px) or `py-24` (96px)
- Mobile: `py-16` (64px)

**Hero Section:**
- Min-height: `100vh` (always)
- Padding: `py-0` (full bleed)

**Compact Sections:**
- Stats bars, trust badges: `py-12` (48px)

---

## Container Widths

**Max Widths:**
- Full content: `max-w-7xl` (1280px)
- Reading content: `max-w-3xl` (768px)
- Narrow content: `max-w-xl` (640px)

**Container Padding:**
- Desktop: `px-8` (32px)
- Mobile: `px-4` (16px)

---

## Component Spacing

**Cards:**
- Padding: `p-6` (24px) or `p-8` (32px)
- Gap between cards: `gap-6` (24px) or `gap-8` (32px)

**Buttons:**
- Padding: `px-6 py-3` (24px × 12px) standard
- Padding: `px-8 py-4` (32px × 16px) large/hero

**Forms:**
- Input padding: `px-4 py-3` (16px × 12px)
- Label margin bottom: `mb-2` (8px)
- Field margin bottom: `mb-6` (24px)

---

## Grid Gaps

**Standard Grids:**
- 2-column: `gap-8` (32px)
- 3-column: `gap-6` (24px)
- 4-column: `gap-4` (16px)

**Tight Grids:**
- Bento grids: `gap-4` (16px)
- Icon grids: `gap-6` (24px)
