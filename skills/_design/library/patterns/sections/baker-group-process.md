# Pattern: Baker Group — Process Timeline & Bold Footer

**File:** `patterns/sections/baker-group-process.md`  
**Reference:** https://www.thebakergroup.com/  
**Use case:** Service businesses with clear process steps, industrial/commercial contractors, businesses wanting strong brand presence

---

## Pattern 1: Process Timeline with Connecting Path

**Visual Structure:**
```
Header: "Proven Process" + description text

[Connected Path Graphic — horizontal flowing line/path]
    ↓              ↓              ↓
[Icon Circle] → [Icon Circle] → [Icon Circle]
   PLAN           BUILD         MAINTAIN
   [desc]         [desc]         [desc]
```

**Key Elements:**
- **3-4 step process** max (Plan → Build → Maintain)
- **Circular icon containers** (100-120px) with white icons on dark background
- **Connecting path/line** between steps — gold/yellow accent color flowing between
- **Bold uppercase labels** below each icon
- **Description text** under labels (2-3 sentences)
- **Curved path** creates visual flow and movement

**Animation (Optional Enhancement):**
- Icons slide in from left one at a time (staggered 200ms delay)
- Path draws itself as icons appear
- Subtle hover: icon scales slightly, path glows

**When to Use:**
- Service businesses with clear process (contractors, agencies, consultants)
- When you want to emphasize methodology/system
- Good for: construction, HVAC, web design agencies, consulting firms

## Pattern 2: Bold Footer with Logo Watermark

**Visual Structure:**
```
┌─────────────────────────────────────────────────┐
│  [Diagonal transition from section above]       │
│                                                 │
│  [HUGE LOGO WATERMARK — 60% opacity, right]     │
│                                                 │
│  [Small Logo]      [Contact Info]              │
│                    24/7 Available              │
│                    Phone (bold)                │
│                    Address                     │
│                                                 │
│  [Social Icons] — circular buttons            │
│                                                 │
│  © Year | Links | Credit                        │
└─────────────────────────────────────────────────┘
```

**Key Elements:**
- **Dark background** (black or near-black)
- **Huge logo watermark** — giant faded logo in background (right side)
- **Diagonal transition** — section above cuts at angle into footer
- **Clean contact stack** — Phone bold/large, address smaller
- **Circular social icons** — unified style, brand colors
- **Minimal text** — only essentials

**Similar to:** Arturos site footer (huge logo treatment)

**When to Use:**
- Strong brand identity businesses
- When logo is clean/recognizable
- Industrial, construction, professional services
- Want to end with brand impact

## Pattern 3: Location Showcase with Building Photo

**Visual Structure:**
```
Header: "Locations"
Subtext: Description of service area

┌──────────────────────┬──────────────────────────┐
│                      │                          │
│   [Building Photo]   │   Location Name          │
│   — exterior shot    │   Address                │
│   — wide angle       │   Phone                  │
│                      │                          │
│                      │   [View on Map button]   │
│                      │                          │
└──────────────────────┴──────────────────────────┘
```

**Key Elements:**
- **Large building exterior photo** — professional, wide shot showing facility
- **Location name bold**
- **Full address + phone**
- **"View on Map" button** — outlined style
- **Clean 2-column layout**

**When to Use:**
- Walk-in businesses (barbers, salons, retail, restaurants)
- Businesses with physical facilities worth showing
- Multiple locations can stack vertically
- Creates trust — "this is a real place"

## Pattern 4: Diagonal Section Transitions

**Visual Effect:**
```
[White/light section above]
        \
         \
          \  ← diagonal line at 15-20° angle
           \
[Dark section below — footer or next section]
```

**Implementation:**
- **SVG or CSS clip-path** creating diagonal edge
- **Color contrast** — light section above, dark section below (or vice versa)
- **15-20° angle** — noticeable but not extreme
- Can use **solid color** or **gradient** along diagonal

**When to Use:**
- Breaking up long scroll
- Transitioning to footer
- Adding visual interest without clutter
- **Caution:** Can look like "crap" on certain sites — use purposefully

## Pattern 5: Detailed Project Portfolio (Blog-Style)

**Structure:**
- **Portfolio grid/list** — project thumbnails
- **Click opens detailed view** — NOT just lightbox
- **Project detail page/popup includes:**
  - Large hero image of completed work
  - Project description (paragraphs)
  - Multiple photos in layout (like blog post)
  - Services provided
  - Results/outcomes
  - Testimonial if available

**Alternative: Project Card Modal**
- Click project → opens modal (not new page)
- Modal contains:
  - Large image carousel
  - Project title + description
  - Key details (location, date, services)
- Close to return to portfolio grid

**When to Use:**
- Construction, renovation, design portfolios
- When projects have stories worth telling
- Showcasing craftsmanship/detail work
- Good for: contractors, architects, designers

## When This Style Works

✅ **Use for:**
- Industrial/commercial contractors
- Professional services with clear process
- Employee-owned or established businesses
- B2B services
- Construction, engineering, manufacturing

❌ **Avoid when:**
- Light/airy brand (yoga, wellness, feminine)
- Simple retail (too heavy)
- Minimal aesthetic (this is bold/intentional)
- Budget/convenience positioning (this says "premium/established")
