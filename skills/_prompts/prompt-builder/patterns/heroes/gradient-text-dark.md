# Hero Pattern: Gradient Text on Dark Background

**Source:** XPO Rehab (xpo-lead-pulse.base44.app)
**Best for:** Healthcare, wellness, modern tech, fitness
**Vibe:** Energetic, modern, bold

## Description
Dark background (solid or subtle gradient) with large headline featuring gradient text effect. Clean, impactful, contemporary feel.

## Layout Structure
```
┌─────────────────────────────────────────────┐
│  [Dark background — solid or subtle grad]   │
│                                             │
│     ╔═══════════════════════════════╗       │
│     ║  GRADIENT TEXT HEADLINE       ║       │
│     ╚═══════════════════════════════╝       │
│                                             │
│     [Supporting subheadline]                │
│     [CTA Buttons]                           │
│                                             │
└─────────────────────────────────────────────┘
```

## Visual Specs

**Background:**
- Dark solid color (near-black, deep navy, charcoal)
- OR subtle dark gradient
- NOT busy — let the text be the focus

**Headline (Gradient Text):**
- Large, bold typography
- Gradient fill: Brand colors (e.g., blue → cyan, purple → pink)
- Can be horizontal or slightly angled gradient
- Single gradient direction (not chaotic)

**Subheadline:**
- White or light gray text
- Standard weight, readable size
- Positioned below headline with proper spacing

**CTAs:**
- Primary: Solid color button (contrasting)
- Secondary: Outline or text link
- Clear hierarchy

**Accent Elements (Optional):**
- Subtle glow behind text
- Decorative shapes (very minimal)
- Floating icons (use sparingly)

## When to Use
- Rehab/healthcare (energy, progress)
- Fitness/wellness
- Tech startups
- Modern service businesses
- When brand colors are vibrant

## Variations
- **Single gradient:** One color to another
- **Multi-stop:** 3+ colors in gradient
- **Animated:** Subtle gradient shift (advanced)
- **With image:** Dark image behind, gradient text on top

## Prompt Snippet
```
Hero Section: Gradient text on dark background
- Background: {DARK_COLOR} solid or subtle gradient
- Headline: "{HEADLINE_TEXT}" 
  - Font: {FONT_FAMILY}, {FONT_SIZE}px, bold
  - Gradient: {COLOR_1} to {COLOR_2} (left-to-right or top-to-bottom)
- Subheadline: "{SUBHEADLINE_TEXT}" — white/light gray, readable
- CTA Primary: "{PRIMARY_CTA}" — {BUTTON_COLOR} solid
- CTA Secondary: "{SECONDARY_CTA}" — outline or text style
- Spacing: Generous padding top/bottom (120px+)
- Optional: Subtle glow effect behind headline text
```

## Notes
- Gradient should use BRAND colors — not random rainbow
- Text must remain readable — don't overdo the gradient complexity
- Dark background makes colors pop — use this to advantage
- Keep supporting elements minimal, let gradient headline dominate
