# Section Pattern: Stats Bar (4-Column with Dividers)

**Source:** General pattern — NOT recommended for portfolio pages per Spencer's feedback
**Best for:** Home services, established businesses, credibility-focused
**Vibe:** Trust, experience, proven results

## Description
Horizontal bar with 4 stat columns separated by vertical dividers. Each stat has large number + label below.

## Layout Structure
```
┌─────────────────────────────────────────────────────────────┐
│                                                             │
│   100%+    │    15+     │    24hr    │    500+              │
│ Satisfaction│  Years    │ Response   │ Projects             │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

## Visual Specs

**Structure:**
- 4 equal columns
- Vertical dividers between columns (subtle lines)
- Centered content in each column

**Numbers:**
- Large, bold typography
- Brand color or dark color
- May include symbols: %, +, hr

**Labels:**
- Smaller text below numbers
- Descriptive (what the number means)
- Regular weight, muted color

**Background:**
- Contrasting section background
- Could be brand color with light text
- Or light background with dark text

## When to Use
- After hero (immediate credibility)
- Before reviews (adds weight to testimonials)
- Home services (years in business important)
- Established businesses with impressive numbers

## When NOT to Use (Per Spencer)
- Bottom of portfolio pages (feels disconnected)
- New businesses without impressive stats
- When it disrupts visual flow

## Variations
- **3 stats:** More space per stat
- **With icons:** Icon above number
- **No dividers:** Spacing only separation
- **Animated:** Count-up animation on scroll

## Prompt Snippet
```
Stats Section: 4-column credibility bar
- Layout: 4 equal columns with vertical divider lines between
- Background: {BG_COLOR} (often contrasting section)
- Each stat:
  * Number: Large ({FONT_SIZE}px), bold, {NUMBER_COLOR}
  * Label: Smaller ({LABEL_SIZE}px), {LABEL_COLOR}, below number
- Stats to include: {STAT_1}, {STAT_2}, {STAT_3}, {STAT_4}
- Examples: "100%+ Satisfaction", "15+ Years Experience", "24hr Response", "500+ Projects"
- Padding: 60px top/bottom
- Mobile: Stack 2x2 or vertical list
```

## Notes
- Numbers must be REAL and IMPRESSIVE — fake stats destroy trust
- This is credibility armor — use when you have credentials to show
- Position matters — usually early in page flow
- Avoid at bottom of pages (Spencer's specific feedback)
