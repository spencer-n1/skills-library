# Wave Shape Divider

**File:** `library/sections/wave-shape-divider.md`

---

## Source
Coastal Garage Doors — coastal-door-flow.base44.app

---

## Best For
- Coastal themes
- Soft transitions between sections
- Light, approachable aesthetics
- Home services, residential focus

---

## Vibe
Soft, flowing transition. Wave shape creates visual interest without being jarring. Suggests movement and flow. Gentler than sharp angles or straight lines.

---

## Visual Specs

**Shape:**
- Curved wave at bottom of section
- Single smooth curve (not multiple ripples)
- Creates transition to next section

**Color Treatment:**
- Solid color wave (full blue, not gradient) ← preferred
- OR subtle gradient if brand allows
- Should match or complement next section's background

**Height:**
- 60-100px wave height
- Not too dramatic (distracting) or too subtle (invisible)

---

## Prompt Snippet

```
Hero section bottom:
- Wave shape divider at bottom
- Solid blue color (not gradient)
- Smooth curve transition to next section
- Height: ~80px
- Creates visual flow between hero and services

OR for section transitions:

Between-section divider:
- Wave shape divider
- Color matches upcoming section background
- Soft, flowing transition
- Maintains coastal/light aesthetic
```

---

## When to Use
- Coastal/blue themed sites
- Want soft transitions (not hard cuts)
- Residential/home services
- Light, approachable brand feel
- Breaking up solid color sections

---

## When NOT to Use
- Dark/urgent themes (wave feels too soft)
- Professional/conservative services (law, finance)
- Modern/minimal sites (wave adds unnecessary decoration)
- Heavy industrial aesthetic
- Mobile-first if wave causes scroll issues
